# Elastic Beanstalk

## Testar a aplicação

    eb init -p docker gerenciador-tarefas

## Iniciar a aplicação

    eb local run --port 5000

## Executar a aplicação

    eb local open

## Criar aplicação na AWS 
    
    eb create gerenciador-tarefas

## Abrir aplicação na AWS

    eb open


## Remover recursos

    eb terminate --all --force
