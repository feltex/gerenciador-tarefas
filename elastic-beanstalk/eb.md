# Elastic Beanstalk

## Testar a aplicação

    eb init gerenciador-tarefas

## Iniciar a aplicação

    eb local run --port 5000

## Executar a aplicação

    eb local open

## Criar aplicação na AWS 
    
    eb create gerenciador-tarefas

## Abrir aplicação na AWS

    eb open

## Atualizar a aplicação na AWS

    eb deploy


## Remover recursos

    eb terminate --all --force
